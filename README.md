# STEEL RUSH − 高速バイクバトル −

高速道路を舞台にした、ブラウザで遊べる3Dバイクバトルゲーム。

[Three.js](https://threejs.org/) を使用したシングルファイル(`index.html`)構成で、GitHub Pages にそのまま公開できます。

## 概要

プレイヤーはバイクに乗った主人公「KAI」を操作し、深夜のハイウェイで追撃部隊と戦う。
左右から迫る敵を斬り、照準攻撃を回避またはガードし、リミットゲージを溜めて必殺技「スピンスラッシュ」を放つ。

## プレイ方法

### PC操作

| キー | アクション |
| --- | --- |
| `W` / `↑` | アクセル |
| `S` / `↓` | ブレーキ |
| `A` / `←` | 左移動 |
| `D` / `→` | 右移動 |
| `J` | 左側を攻撃 |
| `L` | 右側を攻撃 |
| `Space` | ガード |
| `E` / `Enter` | 必殺技 |

### スマホ操作

- 画面左側のボタンで移動・加速・減速
- 画面右側のボタンで攻撃・ガード・必殺技
- タッチ操作に対応しているので、スマホ・タブレットでもそのままプレイ可能です
- **画面が狭いと操作ボタンとHUDが重なりやすいため、横向き(ランドスケープ)プレイを推奨します**

## 特徴

- Three.js による3D表現
- 疾走感のあるハイウェイステージ
- 左右攻撃・ガード・回避・必殺技のアクション
- スマホタッチ操作対応
- WebAudio による簡易サウンド演出
- npm/Vite等のビルド不要、`index.html` のみで完結する静的サイト
- GitHub Pages でそのまま公開可能

## 公開方法(GitHub Pages)

1. GitHubのリポジトリページを開く
2. **Settings → Pages** を開く
3. **Source** を `Deploy from a branch` にする
4. **Branch** を `main`、フォルダを `/ (root)` にする
5. **Save** する
6. 数分後に表示されるURL(例: `https://<ユーザー名>.github.io/steel-rush/`)からプレイ可能
